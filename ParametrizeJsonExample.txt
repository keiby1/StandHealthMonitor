Примеры вызова
// Без дополнительных параметров — подставляются только system и stepString json = TemplatesHolder.getTemplate("A1", "prep");// → {"system": "A1", "step": "prep", "requestId": "${requestId}", "timestamp": "${timestamp}"}// С параметрамиString json = TemplatesHolder.getTemplate("A1", "prep", Map.of(    "requestId", "12345",    "timestamp", "2026-02-08T16:00:00"));// → {"system": "A1", "step": "prep", "requestId": "12345", "timestamp": "2026-02-08T16:00:00"}
Что даёт
Автоматические system и step — не нужны ручные replaceAll.
Единый формат ${var} вместо разных меток.
Один вызов — все замены в одном getTemplate(...).
Без внешних зависимостей — только JDK.
Обратная совместимость — getTemplate("A1", "prep") без третьего аргумента по‑прежнему работает.
Если переменная не найдена в params, плейсхолдер остаётся в строке (например, ${requestId}).
